# Jumadevs Gallery

A photography gallery built collaboratively with Jumadevs.

## Features
- Hero section with smooth scroll
- Image gallery with hover effects
- Dark theme

## Built by
The Jumadevs community.
## Setup

1. Clone the repository.
2. Open index.html directly in a browser, or serve the folder with any static server (for example: npx serve .).
3. app.js wires up smooth scroll, fade-in animations (via animations.css), and the theme toggle.
4. image-loader.js handles lazy-loading images using an IntersectionObserver and a blur-up placeholder.

No build step is required - everything runs as plain HTML/CSS/JS.
